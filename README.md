# code-forge

A Claude Code plugin marketplace for coding assistants.

## Install

/plugin marketplace add your-username/code-forge

## Available plugins

| Plugin | What it does |
|---|---|
| code-reviewer | Deep PR review — bugs, security, performance |
| test-architect | Generate full test suites |
| debug-detective | Root cause analysis and targeted fixes |
| api-crafter | Design and document REST/GraphQL APIs |
| refactor-pro | Detect code smells and apply design patterns |
| standup-scribe | Summarise Slack meeting notes into task lists |
